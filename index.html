📘 دروس HTML الكاملة

الدرس 1: ما هي HTML؟

HTML هي لغة ترميز تُستخدم لبناء صفحات الويب. وهي الأساس لأي موقع. HTML تعني "Hyper Text Markup Language".

<!DOCTYPE html> <html> <head> <title>عنوان الصفحة</title> </head> <body> <p>مرحبًا بك في HTML!</p> </body> </html>

الدرس 2: الوسوم الأساسية

الوسوم (Tags) تُكتب بين < و > وتُستخدم لتحديد أجزاء الصفحة، مثل العناوين، الفقرات، الصور...

  • <p>: فقرة
  • <h1> إلى <h6>: العناوين
  • <img>: صورة
  • <a>: رابط
  • <ul> / <ol>: قائمة

الدرس 3: الروابط والصور

لإضافة روابط وصور داخل HTML نستعمل:

<a href="https://example.com">زر رابط</a> <img src="image.jpg" alt="وصف الصورة">

الدرس 4: القوائم والجداول

القوائم تُستخدم لعرض عناصر بشكل منظم:

<ul> <li>عنصر 1</li> <li>عنصر 2</li> </ul>

أما الجداول:

<table> <tr><th>اسم</th><th>عمر</th></tr> <tr><td>معاد</td><td>12</td></tr> </table>
